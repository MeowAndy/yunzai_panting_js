# 🎨 Yunzai Painting 画图插件

> Yunzai-Bot 画图插件 - 预设画图/自定义画图/次数管理/余额查询/周排行/本地存图

## ✨ 功能一览

| 指令 | 说明 | 权限 |
|------|------|------|
| `#bnn <提示词>` | 纯文生图 | 所有人 |
| `#bnn <提示词> [图片]` | 看图作画（图生图） | 所有人 |
| `#bnn3 <提示词>` | 一次生成多张（最多5张） | 所有人 |
| `#<预设名>` | 使用云端预设画图 | 所有人 |
| `#绘图帮助` | 查看完整帮助 | 所有人 |
| `#绘图查询次数` | 查看群/个人剩余次数 | 所有人 |
| `#排行bnn` | 本周每日作画统计排行 | 所有人 |
| `#查询额度` / `#查余额` | 查询 API 余额 | 所有人 |
| `#绘图更新预设` / `#更新焚决` | 从云端拉取最新预设 | 主人 |
| `#绘图增加次数 <数量> [目标]` | 给群/个人充次数 | 主人 |
| `#绘图查询所有次数` | 查看全服次数账本 | 主人 |
| `#绘图删除次数 [目标]` | 清空某个记录 | 主人 |
| `#绘图删除所有次数` | 清空全服记录 | 主人 |
| `#开启bnn存图` | 开启本地存图 | 主人 |
| `#关闭bnn存图` | 关闭本地存图（默认） | 主人 |

## 🚀 安装

```bash
# 在 Yunzai 根目录执行
cd plugins
git clone https://github.com/MeowAndy/yunzai_panting_js.git painting
# 重启 Yunzai 即可
```

## ⚙️ 配置

编辑 `Painting.js` 顶部配置区：

```javascript
// ========== 角色名配置 ==========
const BOT_NAME = "菲比";  // 修改这里即可替换所有提示文案中的角色名

// ========== 核心配置 ==========
const API_URL = 'https://你的api地址/v1/chat/completions';
const API_KEY = "你的ApiKey";
const MODEL_NAME = "gpt-5.5";

// ========== gpt-image-2 快速生图配置 ==========
const IMAGE_API_URL = "https://你的api地址/v1/images/generations";
const IMAGE_MODEL_NAME = "gpt-image-2";
```

### 角色名配置

只需修改 `BOT_NAME` 一行，所有提示文案中的角色名会自动替换，无需逐个修改。

## 📌 推荐模型

**本插件最佳搭配 `gpt-5.5` 模型进行画图。**

使用 `gpt-image-2` 可能会出现兼容性问题（如 size 参数格式报错等）。

## 💾 本地存图

- 默认**关闭**，不占用磁盘空间
- 主人发送 `#开启bnn存图` 后，生成的图片会保存到 `data/generated_images/` 目录
- 文件名格式：`日期_时间_用户QQ_序号.png`
- 发送 `#关闭bnn存图` 随时关闭

## 📊 周排行

发送 `#排行bnn` 查看本周每日作画统计：
- 合并转发消息，不渲染图片
- 带字符柱状图直观展示
- 显示总计、日均、最高日
